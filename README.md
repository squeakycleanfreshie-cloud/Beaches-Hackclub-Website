# Beaches Hack Club Website
This website was made by me, for the Hack Club that I lead. It explains everything about our club, and how to join the club.

# Language
When I made this, I enhanced my learning skills in HTML & CSS Styling

# Features
This website features:
 - Navigation
 - Footer
 - Hero
     - Custom art
     - Statistics
     - Buttons
     - How It works
 - About Page
     - Who we are section
     - Stats
     - Meet the Leaders
 - Projects Page
     - Hoverable and Clickable Projects
     - Takes you to individualised project page, with names of people and project made
 - Blog Page
     - One blog post when clicked

# My Inspiration
I really wanted a website for my hackclub, so that more members can join, so instead of relying on someone else to do it for me, I finally stopped procrastinating and did it myself.
